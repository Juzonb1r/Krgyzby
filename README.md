# 🎮 PAC-MAN Classic

Кыргызбай - Classic PAC-MAN game built with HTML5 Canvas and JavaScript.

## 🎯 Play Online

**Play here:** https://juzonb1r.github.io/Krgyzby/

Deployed on GitHub Pages for fast global access.

## 🕹️ Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move PAC-MAN |
| Space | Pause game |

## 📋 Game Rules

- Eat all dots to win
- Power pellets temporarily let you eat ghosts (+200 points)
- Regular dots give 10 points
- You have 3 lives
- Avoid ghosts when not in power mode!

## 🏆 Features

- Classic PAC-MAN gameplay
- 4 unique ghosts (Blinky, Pinky, Inky, Clyde)
- Power-ups and scoring
- High score saved locally
- Smooth animations
- Tunnel teleportation

## 🛠️ Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- CSS3
- AWS S3 + CloudFront deployment

## 📦 Deployment

Hosted on AWS:
- S3 bucket for static file storage
- CloudFront CDN for global distribution
